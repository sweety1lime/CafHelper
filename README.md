# CafHelper

Оверлей-помощник для сотрудников госструктур на **Majestic RP**. Открывается поверх GTA по горячей клавише и подсказывает, как действовать в ситуации: на какие статьи ссылаться, что отвечать адвокатам, где не проколоться. Всё работает офлайн — без серверов, API и подписок.

Поддерживаются **все 19 серверов Majestic** — базы выкачаны с официального форума каждого сервера (суммарно ~20 000 статей).

## Возможности

- **Поиск по ситуации** — пишешь «ствол за спиной», «код 6» или «сколько держать в кпз» — получаешь карточку-сценарий с порядком действий, статьями и готовыми фразами. Понимает игровой жаргон (в том числе фразы), опечатки и запросы в английской раскладке (`vbhfylf` → «миранда»). Мусорные результаты прячутся под «Слабые совпадения», под каждой статьёй видно, чем она совпала.
- **Мастер ситуаций** — не знаешь, что печатать? Выбери чипами: «Задержание → Застигнут на месте» — и получи собранный ответ: подсказки, готовые фразы, сценарий со статьями твоего сервера.
- **Сценарии** — 24 пошаговых шпаргалки: от трафик-стопа и ДТП до захвата заложников и выхода под залог. У каждой — «Порядок действий», «На что ссылаться», «Готовые фразы» и «На чём ловят» (ответы душным адвокатам).
- **Кодексы + Законы** — УК, ПК, КоАП, ДК по главам + все законы штата (об оружии, об ордерах, о статусе адвокатов…). Наказания, приоритет розыска, человекочитаемые флаги («Федеральная», «Розыск», «Через суд»). Клик по статье — копирование ссылки вида `УК 12.8 — Название` в буфер для чата.
- **Калькулятор наказаний** — отмечаешь статьи кнопкой «＋», внизу сумма срока и штрафа + готовая строка для рапорта: `УК 12.8, УК 17.5 | срок: до 7 лет`.
- **Быстрые фразы** — Миранда (дословно из ПК твоего сервера!), «требование остановиться», «отойти 10 метров» — копируются в чат одним кликом.
- **Избранное** — звёздочка на статье закрепляет её на вкладке «Поиск».
- **Оверлей** — окно поверх игры (игра в режиме «оконный без рамки»), показ/скрытие по хоткею (по умолчанию `F9`, настраивается), Esc — скрыть, таскается за шапку. Опционально — автоскрытие при клике в игру. При первом запуске — короткий тур по интерфейсу.

## Статус баз по серверам

- **Atlanta** — эталон: тексты сверены вручную, сценарии написаны под реальные номера статей.
- **Остальные 18 серверов** — тексты кодексов и законов настоящие (с форума сервера), но сценарии адаптированы с Атланты полуавтоматически. Ключевые ссылки (Миранда, задержание, обыск, стадии силы, оружие, наркотики, взятка…) сверены по кодексам вручную; остальное помечено в приложении янтарным чипом **«⚠ не сверено»** — увидел такой, проверь статью во вкладке «Кодексы», прежде чем цитировать адвокату.

Законы на серверах реально разные: где-то «пьяный за рулём» — отдельная статья ДК, а где-то её нет вообще; у одних «препятствование журналистам» в УК, у других — в КоАП. Поэтому и нужна сверка живыми игроками — см. «Как помочь» ниже.

## Установка

Готовый установщик — в [Releases](../../releases). Либо собери сам:

```bash
npm install
npm run tauri dev     # запуск для разработки
npm run tauri build   # exe-установщик в src-tauri/target/release/bundle/nsis/
```

Стек: Tauri 2 + React + TypeScript + Vite + Tailwind. Поиск — MiniSearch, полностью офлайн.

## Как помочь (особенно если играешь не на Атланте)

Самое ценное — **сверка сценариев твоего сервера**. Ты знаешь, как на твоём сервере реально работают задержания, какие граммовки у наркотиков и что говорят адвокаты — а я нет. Вот как помочь, по нарастающей:

**Уровень 1 — просто сообщи.** Увидел ошибку (не тот номер статьи, устаревшая граммовка, кривая фраза) — напиши в Discord: **`psychokid1488`**. Укажи сервер, статью/сценарий и что не так. Исправление прилетит всем через обновление базы, приложение переустанавливать не надо.

**Уровень 2 — исправь данные.** Всё, что показывает приложение, лежит в обычных JSON:

```
data/
  servers.json                 # список серверов
  common/synonyms.json         # жаргон -> слова из статей (общий)
  common/wizard.json           # дерево мастера ситуаций (общее)
  adapt-overrides.json         # ручные решения переноса статей между серверами
  servers/<id>/
    criminal.json …            # УК / ПК / КоАП / ДК / законы (сгенерированы с форума)
    scenarios.json             # сценарии сервера (их и надо сверять!)
```

Открой `data/servers/<твой-сервер>/scenarios.json`, найди ссылки с `"unverified": true`, сверь с кодексом сервера (вкладка «Кодексы» или форум) и поправь номер — а флаг убери. Проверь себя командой `node scripts/validate-data.mjs <сервер>` и пришли Pull Request (или просто файл в Discord). Когда весь файл сверен — убери из него `"draft": true`, и плашка «черновик» исчезнет.

**Уровень 3 — новые сценарии и жаргон.** Формат сценария простой (посмотри любой в `scenarios.json`): ситуация → шаги → статьи → фразы → «на чём ловят». Знаешь типовую ситуацию, которой не хватает, — пиши или присылай PR. То же со словарём жаргона: если поиск не понял слово, которым все пользуются, — добавь его в `common/synonyms.json`.

## Как устроен конвейер данных (для любопытных)

Базы не набиты руками — они выкачиваются с форума и собираются скриптами:

1. `node scripts/scrape-forum.mjs <сервер>` — открывает Edge (Cloudflare проходится в окне), выкачивает все темы раздела «Законодательная база» сервера, включая посты-поправки и спойлеры. Ссылки на разделы всех серверов — в `data/forum-sections.json`.
2. `node scripts/convert-all.mjs --only=<сервер>` — распознаёт, какая тема какой кодекс (по заголовку — на разных серверах они называются по-разному), конвертирует в JSON и прогоняет валидатор. Парсер знает четыре семейства форумной разметки и сам определяет нужное.
3. `node scripts/adapt-scenarios.mjs <сервер>` — переносит сценарии Атланты на сервер: ищет каждую статью в кодексах сервера по полному тексту (а не только заголовку), умеет находить нормы, переехавшие в другой кодекс, и честно помечает `unverified` то, в чём не уверен. Ручные решения хранятся в `data/adapt-overrides.json` и переживают перегенерацию.
4. `node scripts/validate-data.mjs --all` — страховка: пороги числа статей, битые ссылки, дубли, связность дерева мастера.

Обновление без релиза: кнопка «Проверить обновления базы» в Настройках сравнивает `manifest.json` этого репозитория с локальной версией и докачивает свежие JSON из `data/`. То есть принятый PR с исправлением базы доезжает до игроков без пересборки приложения — достаточно поднять дату в `manifest.json`.

## Дисклеймер

CafHelper — шпаргалка для ролевой игры, а не юридический справочник. Тексты взяты с официального форума Majestic RP и могут отставать от свежих правок — дата базы видна в Настройках. Финальное слово всегда за администрацией твоего сервера.
